# 💳 Simulador de Pagamentos

Projeto desenvolvido em Java para simular um sistema de processamento de pagamentos utilizando fila.

## 📌 Funcionalidades
- Criação de contas
- Processamento de pagamentos
- Fila de pagamentos
- Simulação de operações financeiras

## 🛠️ Tecnologias
- Java

## ▶️ Como executar
1. Importar o projeto no Eclipse
2. Executar a classe `Main`

## 📂 Estrutura
- `src/` → código fonte
